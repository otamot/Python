# kmeans
k-meansによってクラスタリングを行う。
ここではN次元のベクトルを持つ各ユーザをn個のクラスタに分ける。

## 入力ファイル
inputファイルはCSVの形式で、各業に
label,要素1,要素2,...,要素N
の形式でデータが格納されている。
以下がinputフィルのsampleである。
